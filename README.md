本项目仅供学习使用，不提供任何资源存储，歌曲等数据来自互联网，请自觉遵守相应规则（比如禁止恶意下载盗版歌曲进行商用等等）~

## Usage

- 运行前确保已经运行`网易云API`服务

```
git clone https://gitee.com/tadm/TadmPlayer.git

cnpm install

yarn start

yarn build
```
